To set up an environment on Windows:
$ python -m venv venv
$ venv\Scripts\activate

Install Django

$ pip install pipenv
$ pipenv shell
$ pip3 install shell (if the line above doesn't work)
$ pipenv install django
$ django-admin startproject brewdog_project (only done once!)
$ python manage.py startapp brewdog_project
$ pip install django djangorestframework
$ python manage.py migrate
$ python manage.py runserver

To run backend:
$ python manage.py startapp brewdog_project
$ python manage.py migrate
pythong manage.py runserver

Django should be available at: http://localhost:8000 


Install React

$ pip install django djangorestframework
$ npm install react and node.js
$ npm i webpack webpack-cli --save-dev
$ npm i @babel/core babel-loader @babel/preset-env @babel/preset-react --save-dev
$ npm i react react-dom --save-dev
$ npm install @material-ui/core  
$ npm install --save react@16.8.0 react-dom@16.8.0
$ npm install @babel/plugin-proposal-class-properties
$ npm install react-router-dom

To run frontend:
$ npm run dev
